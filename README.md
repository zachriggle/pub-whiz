Maxs
====

Scraping local bars and Cross-Referencing BeerAdvocate!

Currently Supports:

- Maxs Taphouse
- Frisco Taphouse
- The Ale House Columbia
- Victoria Gastro Pub

Quasi-Supports:

- Kloby's BBQ (requires manual updates due)
    - The tap list is an image QQ

Requirements
====

- Python 2.7
- BeautifulSoup 4
- Jade
- gocr, libjpeg

Usage
====

Clone the repository

    $ source update.sh
